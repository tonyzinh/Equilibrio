Sobre

Nosso jeito de ser, nossa cultura, nossos valores

Equillibrio é uma empresa dedicada ao bem-estar emocional e psicológico de seus clientes, oferecendo uma ampla gama de serviços de consultas psicológicas. Nosso time é formado por psicólogos altamente qualificados e especializados em diversas abordagens terapêuticas, como terapia cognitivo-comportamental, psicanálise, e psicoterapia humanista, garantindo um atendimento personalizado para cada necessidade.

Linguagens

Conheça Algumas Linguagens que Foram Utilizadas

Equillibrio é uma empresa dedicada ao bem-estar emocional e psicológico de seus clientes, oferecendo uma ampla gama de serviços de consultas psicológicas.

HTML

Define a estrutura de uma página web, organizando elementos como textos, imagens, links, tabelas e formulários. É o esqueleto que serve de base para a exibição de conteúdos.

CSS

Responsável pela aparência da página web. Define cores, fontes, tamanhos, posicionamento e espaçamento. Permite criar animações, layouts responsivos e personalizações visuais.

Bootstrap

Um framework que combina CSS e JavaScript, ajudando a criar sites responsivos, com design moderno. Inclui componentes prontos como botões, menus, formulários e layouts flexíveis.

Javascript

Linguagem de programação para criar interatividade em páginas web. É usada para validação de formulários, animações, atualizações dinâmicas e comunicação com servidores.

Java

Linguagem de programação para criar funcionalidades robustas. Usada em sistemas web, aplicativos Android e back-end, interage com bancos de dados e gerencia APIs e lógica de negócios.

MySQL

Banco de dados relacional usado para armazenar e gerenciar informações. Permite criar, consultar, atualizar e deletar dados, sendo amplamente utilizado em sistemas web e aplicativos.

Spring Boot

Framework Java para criar aplicações web robustas e rápidas. Facilita o desenvolvimento com configurações automáticas, APIs REST, integração com bancos de dados e segurança integrada.

Sass

O Sass (Syntactically Awesome Stylesheets) é uma linguagem de extensão do CSS que simplifica a escrita de estilos com recursos como variáveis, aninhamento, mixins e herança, tornando o CSS mais eficiente.
